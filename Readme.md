# AnonTXT demo app

## Getting started

**Setup node-orm2!**


**Setup AnonTXT**

Edit `anontxt/config/settings.js` to set your database, user & password.

```bash
cd examples/anontxt
npm install
edit config/settings
node server.js
```

And then open up [localhost:3000](http://localhost:3000/)

"# anontxt"
